# Goal

* Get info on the status of a door, **open** or **closed** using external LED.
* Control using an Wifi Access Point the door with a web server
* display all relevant informatiopn on a 0.96" oled screen (optionnal)
* debug easily using color

# Hardware

* ESP32-S3or any ESP32 version but you need to adapt the script for PIN
* LED (5V)
* Resistance to protect the LED (using a 270ohms)
* 0.96" Oled I2C screen (aliexpress or any website)
* MC-38 magnetic captor, or any other stuff like this
